また論点に関するコメントの主張・その根拠を、原文の意味や内容をできるだけ省かずに抽出し、一人称視点・丁寧な表現にまとめて出力してください。

$context_block

コメント:
"""
$content
"""

可能な立場: "$stanceOptions"

注意事項:
- "立場なし": コメントが論点に対して明確な立場を示していない場合
- "その他の立場": コメントが論点に対して明確な立場を示しているが、与えられた選択肢のいずれにも当てはまらない場合
- コメントの言外の意味を読み取ろうとせず、明示的に書かれている内容のみを分析してください
- extractedContent はフラットなテキストで、コメントの当事者目線で記してください。根拠が添えられた事実と本人の考えとを注意して書き分けてください

以下のJSON形式で回答してください:
{
  "reasoning": "考察",
  "stance": "立場の名前",
  "confidence": 信頼度（0から1の数値）,
  "extractedContent": "論点についてコメントから抽出した主張とその根拠"
}
